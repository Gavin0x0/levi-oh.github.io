    # 食用方法

    • 你的电脑还需要安装好 Git (opens new window) 和 Node.js (opens new window) (其中包括了 npm (opens new window)) 在命令行中运行：
    # 克隆这个仓库
git clone https://github.com/levi-oh/pic2animation.git

# 进入项目文件夹
cd pic2animation

# 安装依赖
npm install

# 在 localhost:9080 本地启动
npm run dev

# 打包为electron应用
npm run build
